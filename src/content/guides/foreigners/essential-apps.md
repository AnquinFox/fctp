## 🔧 App Setup Tips

### WeChat Setup:

![WeChat App](/images/general/apps-wechat.webp)

1. Download from App Store / Google Play
2. Register with your phone number (international OK)
3. Add a profile photo and name
4. Add friends by scanning their QR code
5. Set up WeChat Pay (see payment guide)

### Alipay Setup:

1. Download from App Store / Google Play
2. Register with phone number
3. Complete identity verification
4. Add your foreign bank card
5. Start using mini-programs (see below)

### Amap Setup:

1. Download from App Store / Google Play
2. Allow location access
3. Set language to English (if available)
4. Download offline maps for your destination
5. Set home/work addresses
